# pydoodle
An API wrapper of online compiler jdoodle.com written in python.


## Features
 - Easy to use.
 - Over 50+ languages to compile.
 - Get the credits spent information.


## How to install?
Use the following command on you device to install it.
```
pip install pydoodle
```
